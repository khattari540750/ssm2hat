ビルド・インストール方法
----------------------
 このプロジェクトは、以下の手順にて、cmake を利用してビルド、インストールします。
 あらかじめ cmake をインストールしておいてください。

    $ git clone http://gitbucket.team-lab.local/git/hattori/libscip2hat.git
    $ cd libscip2hat
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    $ sudo make install
